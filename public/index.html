ServiceMS

Система управления сервисом роботизированного оборудования

ServiceMS помогает вести каталог оборудования, клиентов, ремонтных заявок, сервисных событий и разграничивать доступ между сотрудниками и клиентами.

Основные возможности

  • учёт оборудования и его статусов;
  • работа с клиентами и заявками;
  • планирование сервисного обслуживания;
  • личный кабинет клиента;
  • доступ по ролям.

Для сотрудников

Администраторы, менеджеры и инженеры получают доступ к внутреннему рабочему кабинету с разделами оборудования, клиентов, заявок и обслуживания.

Для клиентов

Клиент видит только своё оборудование, свои заявки и историю обслуживания, не получая доступа к данным других организаций.

Для дипломного проекта

Проект демонстрирует backend API, frontend-интерфейс, авторизацию, роли и связь между пользователями и клиентскими организациями.
